pythia-0.1: A utility program for retrieving system information.

pythia retrieves system information. It is intended to be used with existing utilities the user already has on their system.

The package is organized into self-contained services, where a given service is used to query a particular item of information. For instance, to query battery information, one would use the service defined in Pythia.Services.Battery.

The entry point is Pythia.
