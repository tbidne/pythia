queryMemory :: MemoryConfig -> IO SystemMemory Source #

Queries the memory based on the configuration. If $sel:app:MkMemoryConfig is Many then we try supported apps in the following order:

[MemoryFree]

Throws:

  • FreeParseError
  • CommandException

data RunApp a Source #

Used in conjunction for our services for indicating how we want to run queries. Many is the identity for its monoid instance.

Examples

Expand
>>> Many <> Single "r"
Single "r"

>>> Single "l" <> Many
Single "l"

>>> mempty @(RunApp String)
Many

Since: 0.1

Constructors

Many

Do not specify an application. Attempt to run all, returning the first success.

Since: 0.1

Single a

Runs a single query based on the parameter app.
