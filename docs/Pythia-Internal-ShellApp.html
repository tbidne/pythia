tryAppActions :: [AppAction IO result] -> IO result Source #

Queries for information via multiple apps. Returns the first success. If any errors are encountered or no actions are run (either because the list is empty or none are supported), an exception is thrown.

Throws:

  • NoActionsRunException: if no actions are run (i.e. the list is empty or none are supported).
  • SomeExceptions: if at least one command is run yet there were no successes.

Since: 0.1

tryIOs :: [IO result] -> IO result Source #

Generalized tryAppActions to any IO. Has the same semantics (i.e. returns the first success or throws an exception if none succeeds) without checking for "support".

Throws:

  • NoActionsRunException: if no actions are run (i.e. the list is empty).
  • SomeExceptions: if at least one command is run yet there were no successes.

Since: 0.1

Utilities

runCommand :: Command -> IO Text Source #

Runs a Command and returns either the text result or error encountered. This is used by SimpleShell to run its command before the result is parsed. This function is exported for convenience.

Throws:

  • CommandException: if running the command returns ExitFailure exit code.
